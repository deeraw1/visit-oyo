Visit Oyo

A mobile-first, web-based cultural tourism platform showcasing the heritage and landmarks of Oyo State, Nigeria.
Overview

Visit Oyo redefines cultural tourism through a QR-driven, offline-first experience that celebrates Yoruba heritage. Inspired by premium tourism platforms like Visit Rwanda, we combine modern web technologies with deep cultural localization to make Oyo State's history accessible to everyone.

Zero friction. Scan a QR code, explore instantly. No app required.
Key Features

    QR-First Access - Physical QR codes at landmarks launch directly to detailed place pages
    Bilingual Support - Culturally accurate English/Yoruba content (not robotic translation)
    Audio Narration - Listen to landmark histories while you explore
    Offline-First UX - Works seamlessly with limited connectivity
    Context-Aware AI Chatbot - Ask questions about specific landmarks (powered by Google Gemini)
    Live Weather - Real-time weather data for Ibadan
    orizontal Discovery - Swim through content with smooth carousel interactions

Tech Stack

    Framework: Next.js 14 (App Router)
    Styling: Tailwind CSS
    Animations: Framer Motion
    AI: Google Gemini Pro API
    Maps: Leaflet / Google Maps Embed
    Hosting: Vercel

Design Principles

    Horizontal over Vertical - Swipeable carousels, not endless scrolling
    Contextual Intelligence - AI that knows the landmark you're viewing
    Hybrid Accessibility - Read, listen, or view content your way
    Zero-Friction Entry - No login. No signup. No download.

Getting Started
bash

# Clone the repository
git clone https://github.com/your-org/visit-oyo.git

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env.local
# Add your API keys (Gemini, Weather, Maps)

# Run development server
npm run dev
```

Open website{} to view the platform.
